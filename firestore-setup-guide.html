🔥 Firestore API有効化ガイド

⚠️ 重要な前提条件

プロジェクトID: quiz-e08fc

このガイドは上記のFirebaseプロジェクト用です。

1 Firebase Consoleにアクセス

以下のURLから Firebase Console にアクセスしてください：

https://console.firebase.google.com/
🚀 Firebase Console を開く
2 プロジェクトを選択

Firebase Console で以下のプロジェクトを選択してください：

プロジェクト名: quiz-e08fc
プロジェクト一覧から「quiz-e08fc」をクリック
3 Firestore Databaseにアクセス

左側のメニューから「Firestore Database」を選択します：

  • 左側ナビゲーションの「構築」セクションを開く
  • 「Firestore Database」をクリック
左メニュー → 構築 → Firestore Database
4 データベースの作成

初回の場合は「データベースの作成」ボタンが表示されます：

  1. 「データベースの作成」をクリック
  2. セキュリティルールを選択：
    • 開発用：「テストモードで開始」を選択
    • 本番用：「本番モードで開始」を選択
  3. ロケーションを選択（推奨：asia-northeast1 (東京)）
  4. 「完了」をクリック

🔒 セキュリティルールについて

開発用（テストモード）:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

本番用（推奨）:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /questions/{document} { allow read: if true; allow write: if false; // 読み取り専用 } } }
5 APIキーとプロジェクト設定の確認

Firestore作成後、以下の設定を確認してください：

  1. プロジェクト設定にアクセス（⚙️ アイコン）
  2. 「全般」タブでプロジェクトIDを確認
  3. 「ウェブアプリ」セクションでAPIキーを確認

firebase.ts ファイルに以下の設定が含まれています：

6 完了確認

Firestore Database が正常に作成されると：

  • ✅ Firestore Database ダッシュボードが表示される
  • ✅ 「コレクション」作成ボタンが表示される
  • ✅ データの読み書きが可能になる

🚨 トラブルシューティング

よくある問題と解決方法：

  • プロジェクトが見つからない：
    • 正しいGoogleアカウントでログインしているか確認
    • プロジェクトの所有者・編集者権限があるか確認
  • Firestore Databaseメニューが表示されない：
    • ブラウザを更新してみる
    • Firebase プロジェクトの課金設定を確認
  • データベース作成に失敗：
    • インターネット接続を確認
    • 別のブラウザで試してみる
    • Firebase サポートに問い合わせ
7 次のステップ

Firestore API有効化完了後：

  1. Reactアプリケーションを起動：
    2. npm run dev
